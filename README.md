## ✨ Features

- 🏎️ **Optimized Performance**: Leverages Vite for lightning-fast HMR and minimal bundle sizes.
- 🎨 **Modern Aesthetics**: Glassmorphic UI with a customized Tailwind CSS v4 design system.
- 🎭 **Engaging Animations**: Complex entrance and layout transitions powered by Framer Motion.
- 🖱️ **Premium UX**: Smooth scrolling experience via Lenis and custom cursor interactions.
- 📊 **Dynamic Components**: Automated timeline, interactive project cards, and categorized skills.
- 📱 **Fully Responsive**: Seamless experience across mobile, tablet, and desktop screens.

## 🛠️ Tech Stack

### Core
- **React 19**: Modern UI development with the latest React features.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS v4**: Utility-first styling with advanced CSS capabilities.

### Libraries & Tools
- **Framer Motion**: Production-ready motion library for React.
- **Lenis**: High-performance smooth scroll library.
- **React Icons**: Comprehensive icon sets (Font Awesome 6, etc.).
- **clsx & tailwind-merge**: Dynamic class management.

## 📂 Project Structure

```text
src/
├── assets/             # Static images, icons, and vectors
├── components/         # React components hierarchical structure
│   ├── ui/             # Atomic UI components (Buttons, Inputs, Titles)
│   ├── Hero.jsx        # Landing section with animations
│   ├── About.jsx       # Personal profile and background

│   └── Contact.jsx     # Interactive contact form
```

## ⚡ Getting Started

Ensure you have [Node.js](https://nodejs.org/) (v18+) installed on your machine.

### Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/01mayankk/portfolio.git
   ```
2. **Install NPM packages**
   ```bash
   npm install
   ```
3. **Start the development server**
   ```bash
   npm run dev
   ```

### Scripts

- `npm run dev`: Starts the local development server.
- `npm run build`: Compiles the application for production.
- `npm run lint`: Perfroms ESLint checks.
- `npm run preview`: Locally previews the production build.

## 🌐 Deployment

This project is optimized for deployment on platforms like **Vercel**, **Netlify**, or **GitHub Pages**.

### Deploy to Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project root.
